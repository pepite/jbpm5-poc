This is the POC REST API documentation.

/admin/process-definitions/list - List all the available workflow

/admin/process-instances/{name}/start - Start a process instance for the given workflows

/admin/process-instances/list - List all the running process instances

/admin/process-instances/{id} - Details the given process instance

/admin/process-instances/{id}/node-name/{name}/retry - Retry the activity with name {name} for the given process instance

/admin/process-instances/{id}/node-id/{idnode}/retry - Retry the activity with id {id} for the given process instance

/admin/tasks/{name}/list - Give all the tasks assign to user {name}

/admin/tasks/{id}/complete
Complete the task with id {id}
