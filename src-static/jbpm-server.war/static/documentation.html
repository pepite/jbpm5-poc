This is the POC REST API documentation.

Url Description
/admin/process-definitions/listList all the available workflow
/admin/process-definitions/list Start a process instance for the given workflows
/admin/process-instances/list List all the running process instances
/admin/process-instances/{id} Details the given process instance
/admin/process-instances/{id}/node-name/{name}/retry Retry the activity with name {name} for the given process instance
/admin/process-instances/{id}/node-id/{idnode}/retryRetry the activity with id {id} for the given process instance
/admin/tasks/{name}/listGive all the tasks assign to user {name}
/admin/tasks/{id}/completeComplete the task with id {id}
/admin/process-instances/{id}/metadata/imageProvide image metadata for process instance {id}
